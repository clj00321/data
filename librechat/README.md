## XPath Dokumentation

| XPath | Synlig Tekst | Hyperlink | Kommentar |
|-------|--------------|-----------|---------|
| `//*[@id="main"]/div[1]/h2` | Nyheder | [Se alle nyheder](https://eksempel.dk/nyheder) | Hovedoverskrift |
| `//a[contains(@href, "/produkt/")]` | Produktnavn | [Produktlink](https://eksempel.dk/produkt/123) | Dynamisk produktlink |
| `//img[@class="product-image"]/@src` | - | - | Billed-URL |
